# CRAN-implementation
Cloud Radio Access Network (CRAN) Implementation for LoRa 

# Windows (CMD)
```
python -m venv cran
cran\Scripts\activate
pip install -r requirements.txt
```

# Windows (Git Bash)
```
python -m venv cran
source cran/Scripts/activate
pip install -r requirements.txt
```
# Mac OS
```
python -m venv cran
source cran/bin/activate
pip install -r requirements.txt
```
